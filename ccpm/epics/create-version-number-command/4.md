# Task: Create version.sh bash script

## Description

Create the bash script that reads the VERSION file and outputs the version number in a user-friendly format. This script will be called by the `/pm:version` slash command.

## Acceptance Criteria

- [ ] File created at `claude-template/scripts/pm/version.sh`
- [ ] Script has proper shebang (`#!/bin/bash`)
- [ ] Reads VERSION file from `claude-template/VERSION`
- [ ] Outputs format: `CCPM version X.Y.Z.N`
- [ ] Handles missing VERSION file with clear error message
- [ ] Script is executable (`chmod +x`)
- [ ] File is committed to git

## Technical Details

**File Location**: `claude-template/scripts/pm/version.sh`

**Script Implementation**:
```bash
#!/bin/bash
VERSION_FILE="claude-template/VERSION"

if [[ -f "$VERSION_FILE" ]]; then
  VERSION=$(cat "$VERSION_FILE")
  echo "CCPM version $VERSION"
  exit 0
else
  echo "❌ VERSION file not found. Please reinstall CCPM templates."
  exit 1
fi
```

**Implementation Notes**:
- Use relative path `claude-template/VERSION` (works from repo root)
- Test for file existence with `[[ -f "$VERSION_FILE" ]]`
- Use `cat` to read single line (simple and fast)
- Exit 0 on success, exit 1 on error
- Error message should be helpful for users

**Files Affected**:
- Creates: `claude-template/scripts/pm/version.sh`

**Error Handling**:
- Missing VERSION file → Clear error message
- Empty VERSION file → Will display "CCPM version " (acceptable for this simple implementation)
